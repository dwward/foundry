                                                    Migrate from foundry_ml [planned deprecation] to palantir_models

                                                    Planned deprecation

                                                    The foundry_ml library and dataset-backed models have entered the Python 3.9 deprecation in October 2025 . Support remains available until the deprecation date. At this time, you should migrate your workflows to use the palantir_models library, either by training a new model or by how to train a model in Code Repositories or

                                                    From version 0.1599.0 onwards, the palantir_models library offers better experience with Live Inference . Separately, we recommend replacing experiments where possible. Note that experiments now support chart metrics.
